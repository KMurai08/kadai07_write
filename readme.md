# ① 課題番号-プロダクト名

読書感想保存サイト

## ② 課題内容（どんな作品か）

- 入力した感想のタイトルをリンクとして一覧表示して、リンク先で本文や書籍名などの詳細を確認できる
- 記事の編集削除ができる（7/4追記）
- ユーザー登録機能実装。ゲストユーザー時は記事の詳細は確認できない（7/11追記）

## ⑤ 工夫した点・こだわった点

- インプットに入力した内容を表示するのに、一旦リンクを挟んだ
- textareaにはvalueの値が渡せなかったのでjavascriptで変換して表示した（7/4追記）
- ゲストユーザーの時には記事作成ボタンを非活性にした(7/11)
